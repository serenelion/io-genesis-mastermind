# Archive - Old Website Versions

This folder contains previous iterations of the Integrative Organizing marketing site that are no longer in use.

## Archive Date
February 7, 2026

## Why Archived
These files represent older design systems and copy approaches that were replaced with the current indigo-gold brand system focused on village creator value.

## Contents

### old-html/
- `index_genesis.html` - Early "Genesis Event" branding (eco-luxury cream design)
- `index_v2.html` - Second iteration with TerraLux ecosystem focus

### old-scripts/
- `genesis.js` - JavaScript for genesis version
- `main.js` - Generic main script
- `main_genesis.js` - Genesis-specific interactions
- `main_v2.js` - Second iteration scripts

### old-styles/
- `eco-luxury.css` - Cream/warm-linen/gold design system (Aesop-inspired)
- `main.css` - First main stylesheet
- `main_v2.css` - Second iteration styles
- `schedule.css` - Standalone schedule component styles

## Current Production Files
The active marketing site uses:
- `/index.html` - Main page
- `/src/styles/indigo-gold.css` - Indigo + gold brand system
- `/src/assets/logo.png` - Integrative Organizing logo
- `/netlify.toml` - Deployment configuration
- `/README.md` - Project documentation
- `/DEPLOY.md` - Deployment guide

## Recovery
If you need to reference or restore any of these files, they are preserved here in their original state.

Do not delete this archive folder without backing up to version control.
