# Integrative Organizing: Village Creator Mastermind

Integrative Organizing is a mastermind experience design and facilitation unit of TerraLux Living. The Village Creator Mastermind is a 3-day immersive for community creators expanding regenerative villages.

## Event Details
- **Dates:** June 26-28, 2026
- **Location:** Galway Wellness Club & Big Hollow Greene, Windham NY
- **Participants:** 16 max (invitation-only)
- **Price:** $3,300 per person (50% off for TerraLux Living members)
